# MongoChatting

## 몽고디비로 만든 채팅 웹 


 # 프로젝트 제목 

  &nbsp;&nbsp; &nbsp;  MongoChatting  
  
  <br/>
  
 # 프로젝트 개요
   <p>
   &nbsp;&nbsp; &nbsp;  몽고디비로 만든 채팅 웹 입니다. </p>
   <p> &nbsp;&nbsp; &nbsp;  몽고디비의 특징인 여러 데이터를 조건 없이 insert 하는 것을 시도해봤습니다. </p>
   <p> &nbsp;&nbsp; &nbsp;  동적인 것들은 자바스크립트로 제작해서 자바스크립트 학습도 같이 했습니다.</p>
   
  <br/>
